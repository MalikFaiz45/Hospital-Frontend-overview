Hospital Management System - IPO Chart

Input

- Patient ID, Name, Age, Disease
- Phone Number (for appointment)
- Doctor choice
- Menu choice for actions (Register, Review, Book, Delete)

Processing

Pseudocode:

1. Read patient/appointment details from user
2. IF registering patient THEN
       - Assign Department using autoDepartment(disease)
       - Assign Emergency using autoEmergency(disease)
       - Assign Referral using autoReferral(disease)
       - Calculate DoctorFee and MedicineFee
       - Calculate TotalFee
       - Store patient data in array
       - Save to file
3. IF booking appointment THEN
       - Choose doctor and timing
       - Store appointment in array
       - Save to file
4. IF viewing or searching THEN
       - Read data from file
       - Display requested records
5. IF deleting patient THEN
       - Find patient by ID or Name
       - Remove from array
       - Update file

Output

- Confirmation messages (e.g., "Patient Registered Successfully", "Appointment Booked")
- Patient records displayed in console
- Appointment details displayed
- Error messages for invalid input
- Updated patient file and appointment file
