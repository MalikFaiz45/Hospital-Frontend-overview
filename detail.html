Hospital Management System

Programming Concepts in C++

1. Structs

Structs are used to group related data together into one entity.
In our hospital system, we use person for patients and appointment for online appointments.

Example of patient struct 
struct person {
    int id, age;
    string name, disease;
    string department;
    string emergency;
    string referral;
    string appointment;
    int doctorFee;
    int medicineFee;
    int totalFee;
};

2. Functions

Functions are reusable blocks of code that perform a specific task.
They improve code organization, readability, and reduce repetition.

Example: Automatic Department Function 
string autoDepartment(string disease) {
    if(disease=="fracture" || disease=="heart attack" || disease=="kidney failure")
        return "ICU";
    else
        return "OPD";
}

3. Automatic Functions

Automatic functions calculate values based on input data.
For example, determining emergency status, referral hospital, or calculating fees automatically.

Examples of automatic functions 
autoEmergency(disease) -> YES/NO
autoReferral(disease) -> Returns referral hospital if needed
calculateDoctorFee(department) -> Returns fee
calculateMedicineFee(disease) -> Returns fee

4. File Input / Output

File I/O allows saving patient and appointment records permanently.
Functions like saveAllPatientsToFile() and readFromFile() help store and retrieve data.

5. Arrays

Arrays store multiple values of the same type.
Example: person patient[MAX] stores all patient records.

6. Loops

Loops are used to repeat a block of code.
Example: Looping through all patients to display their details.

7. Conditional Statements

Conditional statements (if/else, switch) are used to make decisions.
Example: Selecting doctor based on user choice, checking disease type, or emergency status.

8. Main Menu & Function Calls

The main function uses a menu system to call different functions:

  • Register Patient
  • Review Records
  • Book Online Appointment
  • View Appointments
  • Search Patient
  • Delete Patient
This demonstrates the use of loops, switch-case, and function calls.
